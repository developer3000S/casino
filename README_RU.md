# BunnyBet Казино
Веб-приложение казино, разработанное с использованием Node.js/Express, React.js/Redux и HTML Canvas

## Описание проекта

BunnyBet - это полнофункциональное онлайн-казино с множеством азартных игр. Проект включает в себя серверную часть на Node.js и клиентскую часть на React.js, соединенные через REST API.

### Основные возможности:
- Игра в рулетку с реалистичной физикой вращения
- Блэкджек (21) с различными правилами
- Слот-машины с анимациями и выигрышными комбинациями
- Крэпс (бросание костей)
- Покер (5-card draw и Texas Hold'em)
- Баккара
- Кено
- Ставки на скачки

### Технологии:
- **Backend**: Node.js, Express.js
- **Frontend**: React.js, Redux для управления состоянием
- **Графика**: HTML Canvas для рендеринга игровых столов
- **База данных**: (указать используемую БД)
- **Аутентификация**: JWT токены
- **Платежи**: Интеграция Stripe и Google Pay

## Безопасность

Все секретные ключи API (включая Stripe) должны храниться в переменных окружения и никогда не коммититься в репозиторий.

### Переменные окружения:
- `STRIPE_SECRET_KEY` - Секретный ключ Stripe для обработки платежей
- `JWT_SECRET` - Секрет для подписи JWT токенов
- `DB_CONNECTION` - Строка подключения к базе данных

## Разработка

### Архитектура приложения

Серверная часть следует архитектуре REST API с четким разделением на слои:
1. **Маршруты (Routes)** - определяют конечные точки API
2. **Контроллеры (Controllers)** - содержат бизнес-логику
3. **Модели (Models)** - работают с базой данных
4. **Промежуточное ПО (Middleware)** - обрабатывают запросы до контроллеров

### Игровая механика

Все игры реализованы с использованием HTML Canvas для рендеринга графики и JavaScript для игровой логики. Каждая игра имеет:
- Реалистичную физику (для рулетки и костей)
- Честную систему генерации случайных чисел
- Поддержку многопользовательской игры в реальном времени

## Платежная система

Интегрирована система платежей Stripe с поддержкой:
- Кредитных и дебетовых карт
- Google Pay
- Автоматической обработки транзакций
- Безопасного хранения платежной информации
